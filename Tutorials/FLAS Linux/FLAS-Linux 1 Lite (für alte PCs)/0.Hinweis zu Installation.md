## Hinweis

Bitte beachten Sie, dass die Tutorials für die normale **FLAS-Linux-Version** erstellt wurden.  
Es kann daher vorkommen, dass einige Schritte in anderen Versionen nicht exakt gleich funktionieren.

Bei Fragen können Sie sich jederzeit an uns wenden.

## FLAS-Browser starten

Um den FLAS-Browser zu starten, geben Sie bitte einmalig folgendes Passwort ein:

flas

Dieser Schritt ist nur beim ersten Start erforderlich.  
Ein zusätzlicher Terminalbefehl ist danach nicht mehr notwendig.
